To run the sample:
	1) execute the  database.sql  file from the Plethora.Workflow project in a sample database.
	2) execute the  static.sql  file from the Plethora.Workflow.Sample project in the database.
	3) change the connection string in the Plethora.Workflow.Sample project Program.cs file.
	4) run the Plethora.Workflow.Sample project.
	5) execute the  start_workflow.sql  file from the Plethora.Workflow.Sample project in the database to initiate a new workflow.
